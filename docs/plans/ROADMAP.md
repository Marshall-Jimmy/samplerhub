# Jima's SamplerHub 改进路线图

## 第一期：稳定性与性能优化（高优先级）

### 1.1 架构层修复

| 问题 | 影响 | 方案 | 状态 |
|------|------|------|------|
| preload 路径硬编码 | 打包后崩溃 | 已修复，需验证生产环境 | ✅ 待验证 |
| better-sqlite3 原生模块 | 打包复杂、跨平台难 | 评估迁移至 libsql 或预构建方案 | ⬜ 待评估 |
| asarUnpack 配置 | 安全扫描误报 | 精简 unpack 范围，仅含 .node | ✅ 已完成 |
| 主进程异常未捕获 | 静默崩溃 | 添加 `process.on('uncaughtException')` 日志 | ✅ 已存在 |

### 1.2 数据库性能

| 问题 | 影响 | 方案 | 状态 |
|------|------|------|------|
| FTS5 同步触发器 | 大数据量时 INSERT 慢 | 改为批量重建 FTS 索引，异步后台任务 | ✅ 已完成 |
| 波形数据 blob | 200点×4字节×10万条 ≈ 80MB | 迁移至独立文件存储，数据库只存路径 | ✅ 已完成 |
| 缺失索引 | 查询慢 | 添加 `samples(bpm)`、`samples(key)`、`samples(duration)` 复合索引 | ✅ 已完成 |

### 1.3 扫描性能

| 优化项 | 现状 | 目标 | 状态 |
|--------|------|------|------|
| 解析并发 | 4 并发解析 | Worker Thread 池解析元数据 | ⬜ 待实施 |
| 批量写入 | 100 条批量插入 | 事务批量写入（500-1000条/批次） | ✅ 已完成 |
| 增量扫描 | 全量扫描 | 基于 mtime 快速跳过未变更文件 | ✅ 已完成 |

### 1.4 内存与渲染

| 问题 | 影响 | 方案 | 状态 |
|------|------|------|------|
| 虚拟列表波形视图 | 每行 64px，大数据量卡顿 | 优化 react-window 配置 | ⬜ 待实施 |
| Howler 实例泄漏 | 内存占用 | 验证预加载机制清理逻辑 | ✅ 已完成 |
| Canvas 波形重绘 | 200 条柱状图每帧重绘 | 离屏 Canvas 缓存静态波形 | ✅ 已完成 |

## 第二期：核心体验升级（中优先级）

### 2.1 音频引擎升级

| 功能 | 现状 | 目标 |
|------|------|------|
| 波形显示 | 200 点简化波形 | 完整波形 + 峰值包络，支持缩放 |
| 播放精度 | Howler html5 模式精度低 | 可选 Web Audio API 模式（低延迟） |
| 音高检测 | 仅文件名解析 | 集成 aubio/essentia 实时分析 |
| BPM 检测 | 仅文件名解析 | 可选音频指纹 BPM 检测 |
| 格式支持 | WAV/MP3/FLAC 等常见格式 | 增加 AIFF、OGG、WMA |

### 2.2 搜索与发现

- **语义搜索**：基于音频特征（频谱质心、过零率）的相似度搜索
- **颜色标签系统**：支持自定义颜色 + 图标
- **智能文件夹**：保存搜索条件为动态虚拟文件夹
- **搜索建议**：输入时自动补全标签/分类/BPM/Key

### 2.3 UI/UX 改进

- **缩略图预览**：采样卡片显示频谱图/波形缩略图
- **拖拽排序**：播放列表支持拖拽重排
- **多窗口支持**：详情页可弹出独立窗口
- **主题系统**：支持自定义主题导入/导出
- **快捷键可配置**：用户自定义键位

### 2.4 数据管理

- **自动备份**：数据库定期备份到用户目录
- **云同步接口**：预留 Dropbox/OneDrive/iCloud 同步钩子
- **采样包管理**：识别并分组采样包（如 Cymatics、Splice 包）
- **使用统计**：采样使用频率热力图，帮助发现"僵尸采样"

## 第三期：高级功能与生态（低优先级）

### 3.1 DAW 深度集成

- **VST/AU 插件**：开发 SamplerHub 插件，DAW 内直接浏览库
- **ReWire 替代**：现代 DAW 的 ARA/CLAP 扩展支持
- **MIDI 映射**：采样映射到 MIDI 键盘，快速试听
- **Tempo 匹配**：自动匹配工程 BPM，实时时间拉伸

### 3.2 AI 辅助功能

- **自动标签**：基于音频内容的 AI 分类（鼓/贝斯/合成器等）
- **相似采样推荐**：基于频谱特征的余弦相似度
- **智能重命名**：根据内容建议规范文件名
- **空白检测**：自动标记采样起止点，去除前后空白

### 3.3 协作与分享

- **采样分享**：生成分享链接，打包采样 + 元数据
- **团队库**：多用户共享采样库，权限管理
- **版本控制**：采样迭代版本管理（v1, v2...）

### 3.4 性能极限优化

- **GPU 加速波形**：WebGL 渲染大规模波形
- **内存映射文件**：大采样库懒加载
- **多进程架构**：解析/分析任务完全隔离到 Worker

## 优先级矩阵

```
紧急且重要（P0）          重要不紧急（P1）
├─ 打包稳定性验证        ├─ 波形引擎升级
├─ 数据库索引优化        ├─ 语义搜索
├─ 异常捕获与日志        ├─ 主题系统
└─ 扫描性能优化          └─ 快捷键配置

紧急不重要（P2）          不紧急不重要（P3）
├─ 格式支持扩展          ├─ VST 插件
├─ 搜索建议              ├─ 云同步
└─ 缩略图预览            └─ AI 自动标签
```
